## 📍 Project OverView 📍

MERN stack의 basic concept 을 이해하고 활용하기 위해 진행한 프로젝트로,

✨ 데모 링크 ✨ <br />

### 사용 기술

- HTML5 / CSS3
- JavaScript(ES6+)
- React
- Node.js
- Express
- MongoDB, Mongoose
- SCSS

### 구현 기능

- 아이템 검색
- 전체 아이템 조회
- 사용자 아이템 추가

### ➕ 추가 예정

- 모바일, 태블릿, 데스크탑 반응형 작업
